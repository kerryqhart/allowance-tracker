**Core Components:**
- Transaction table layout and row buttons
- UI components tab area

**Goal System:**
- Goal progress bar layout and section containers
- Goal renderer top/graph/circular sections

All UI layouts maintain the same visual behavior while using the updated
egui API. No functional changes, only API migration for future compatibility.
